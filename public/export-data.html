  • CSV 형식: 엑셀로 쉽게 열어볼 수 있고, 데이터 분석에 적합합니다
  • JSON 형식: 프로그래밍으로 데이터 처리 시 편리합니다
  • 익명 데이터: 개인정보 없이 학습 기록만 포함됩니다

📄 상세 답안 내역 (CSV)

포함 내용: 문제번호, 과목, 정답 여부, 시간 등

📊 학습 기록 요약 (CSV)

포함 내용: 날짜, 정답률, 과목별 통계 등

💾 전체 데이터 (JSON)

포함 내용: 상세 답안 + 학습 기록 + 통계
