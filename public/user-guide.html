🚀 시작하기

1

분야 선택

메인 페이지에서 "새로 시작하기" 버튼을 클릭하고, 재난안전 / 산업안전 / 소방안전 중 선택합니다.

2

시험 선택

선택한 분야의 시험 목록을 확인하고, 준비 중인 시험을 선택합니다. "데이터 있음" 뱃지가 있는 시험만 학습 가능합니다.

3

회차 선택

년도별로 정리된 회차 목록을 확인하고 원하는 회차를 클릭합니다.

4

학습 시작

"학습 시작" 버튼을 클릭하여 대시보드로 이동합니다.

✨ 주요 기능

D-Day 설정

대시보드 상단 초록색 박스의 시험명을 클릭하여 시험 날짜를 설정할 수 있습니다.

표시 형식:
  • 시험 전: D-15 (15일 남음)
  • 시험 당일: D-DAY
  • 시험 후: D+3 (3일 지남)

  • 완료한 문제: 실제로 푼 문제 수
  • 정답률: 전체 정답률 (%)

📖 학습 모드

1. 전체 문제 풀이

추천 대상: 체계적 학습을 원하는 수험생
  • 선택한 회차의 모든 문제 출제
  • 순서대로 진행
  • 중단 시 이어하기 가능
  • 진도율 자동 저장

2. 오답노트 복습

추천 대상: 취약 부분 집중 보완
  • 이전에 틀린 문제만 출제
  • 반복 학습으로 약점 보완
  • 3회 연속 정답 시 오답노트에서 제거 권장
정기적으로 복습하여 오답노트를 비우는 것이 목표입니다!

3. 랜덤 20문제

추천 대상: 빠른 실력 점검
  • 전체 문제에서 무작위 20개 선정
  • 짧은 시간에 학습 가능 (약 15~20분)
  • 다양한 유형 경험

4. 타이머 모드

추천 대상: 실전 연습
  • 실제 시험과 동일한 제한 시간
  • 긴장감 있는 학습 환경
  • 시간 관리 연습
  • 일반적으로 문제당 1~2분 제한

5. 북마크 복습

추천 대상: 중요 문제 재확인
  • 별표 표시한 문제만 출제
  • 헷갈리는 문제 반복 학습

6. 과목별 학습

추천 대상: 특정 과목 집중 학습
  • 과목별로 문제 필터링
  • 약점 과목 집중 공략

예시 과목: 안전관리론, 산업심리 및 교육, 기계위험방지기술, 전기위험방지기술

복습 강화 (1~2주)

  • "오답노트 복습" 매일 진행
  • 북마크한 중요 문제 재확인

실전 대비 (시험 1주 전)

❓ 자주 묻는 질문 (FAQ)

Q1. 학습 시작 버튼이 안 눌려요

원인: 회차를 선택하지 않았거나, JavaScript 오류

해결 방법:
  1. 회차 카드를 다시 클릭
  2. 페이지 새로고침 (F5)
  3. 브라우저 캐시 삭제 후 재접속
  4. 다른 브라우저로 시도

Q2. 분야 선택 화면이 비어있어요

원인: exams-config.js 로딩 실패 또는 인코딩 오류

해결 방법:
  1. 브라우저 캐시 삭제
  2. 강력 새로고침 (Ctrl + Shift + R)
  3. 브라우저 콘솔(F12) 확인 후 에러 메시지 확인

Q3. 통계가 이상하게 표시돼요

원인: localStorage 데이터 손상

해결 방법:
  1. "설정 변경" 후 재선택
  2. 문제 지속 시 "모든 데이터 삭제" 후 재시작

Q4. 다른 기기에서도 학습하고 싶어요

현재 상태: 기기 간 동기화 미지원

임시 해결: 한 기기에서만 학습하고 통계는 수동으로 기록
향후 계획: 데이터 내보내기/가져오기 기능 추가 예정

Q5. 문제에 오류가 있어요

대응 방법:
  1. 해당 문제 번호와 내용 기록
  2. 북마크로 표시
  3. 관리자에게 문의

Q6. D-Day가 업데이트되지 않아요

해결 방법:
  1. 시험명(초록 박스) 다시 클릭
  2. 날짜 재설정
  3. 페이지 새로고침

학습 데이터 관리

⚠️ 중요: 현재 시스템은 브라우저의 localStorage를 사용합니다.

이미지가 포함된 문제

일부 문제는 도표, 그림, 복잡한 수식 등의 이미지를 포함합니다.

이미지 표시 방식

  • 방식 1: 직접 이미지 URL — Question_image 열에 실제 URL이 있으면 문제 하단에 자동 표시, 실패 시 PDF 열기 버튼 제공
  • 방식 2: "O" 표시 (PDF 참조) — 원본 PDF를 확인해야 하며 알림 박스와 PDF 열기 버튼이 나타납니다
PDF 보기 절차
  1. 🔗 PDF 열기 버튼 클릭 → 새 탭에서 Google Docs Viewer 열림
  2. 현재 문제 번호 기준으로 해당 페이지 확인
  3. 팝업 차단 시 URL이 클립보드에 복사됩니다

PDF 파일 규칙

  • 파일명 예: 건설안전기사_2022_1회.pdf
  • GitHub Releases에 업로드되어 있으며 필요 시 직접 다운로드 가능합니다

수식 표시 (MathJax)

LaTeX 형식의 수식이 자동 렌더링됩니다. 예: $x^2 + y^2 = r^2$

🧩 문제 풀이 화면
